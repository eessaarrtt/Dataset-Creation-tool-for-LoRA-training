**Automated dataset creation for LoRA model training!**

## What is it?

A Python script that automatically creates image datasets with prompts and captions for LoRA model training. A Make.com workflow equivalent that runs locally.

## How it works?

1. **Upload reference images** (2+ photos of character's face)
2. **Upload sample images** (full scenes to process)
3. **AI analyzes images** and generates detailed prompts (Gemini or OpenAI)
4. **Wavespeed creates new images** based on prompts
5. **OpenAI generates captions** for LoRA training
6. **Get ready-to-use dataset** in zip format with images and captions

## What you need?

- Python 3.8+
- API keys: Gemini/OpenAI (for prompts), Wavespeed (for images), OpenAI (for captions)
- Reference and sample images
